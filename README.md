# github-copilot-course

![GitHub Copilot](https://img.shields.io/badge/GitHub_Copilot-000?style=flat-square&logo=githubcopilot&logoColor=white)
![VS Code](https://img.shields.io/badge/VS_Code-007ACC?style=flat-square&logo=visualstudiocode&logoColor=white)

Interactive GitHub Skills course for learning AI-assisted coding with GitHub Copilot.

## What You'll Learn

- Setting up GitHub Copilot in VS Code
- Using Copilot for code generation and completion
- Writing effective prompts for better suggestions
- Working with Copilot in GitHub Codespaces

## Prerequisites

- GitHub account
- VS Code with GitHub Copilot extension
- GitHub Copilot subscription (free for students)

## How to Start

1. Fork this repository
2. Open in GitHub Codespaces or VS Code
3. Follow the step-by-step instructions in each module
